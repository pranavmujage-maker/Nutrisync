# 🥗 NutriSync

An AI-powered nutrition tracking and personalized diet recommendation mobile app built using **Expo (React Native)**.

---

## 🚀 Features

- 📸 Food tracking (image-based / manual entry)  
- 🤖 AI-powered nutrition analysis  
- 📊 Daily calorie & macro tracking  
- 🥗 Personalized diet recommendations  
- 📱 Cross-platform mobile app (Android & iOS)  
- 🔥 Smart health insights dashboard  

---

## 🛠 Tech Stack

- React Native (Expo)  
- JavaScript / TypeScript  
- Node.js (if backend used)  
- Firebase / Database (if used)  
- AI APIs for nutrition analysis  

---

## 📦 Installation

### 1. Install dependencies
```bash
npm install
````

---

### 2. Start the app

```bash
npx expo start
```

---

## 📱 Run Options

* Open in Expo Go (scan QR code)
* Run on Android Emulator
* Run on iOS Simulator
* Run on Web
